Actualizar Paquetes de Ubuntu

Deberás escribir el siguiente codigo: sudo apt update, darás ENTER para buscar las actualizaciones y te pedirá tu contraseña del usuario

Para instalar los paquetes deberás escribir lo siguiente: sudo apt upgrade, darás ENTER y comenzaran a cargaran los paquetes que se pueden actualizar, te pedira una confirmacion para saber si quieres actualizar los paquetes.
